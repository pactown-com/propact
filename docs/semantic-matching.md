# Semantic Endpoint Matching

Propact uses semantic matching to automatically find the best API endpoint for your markdown content using sentence-transformers embeddings and LLM-enhanced descriptions.

## 🧠 How It Works

1. **Intent Extraction**: Analyzes markdown headers, content, and code blocks to understand the user's intent
2. **Endpoint Discovery**: Parses OpenAPI specs to extract all available endpoints with descriptions
3. **Semantic Similarity**: Computes cosine similarity between intent and endpoint descriptions
4. **Top-K Selection**: Returns the best matching endpoints with confidence scores

## 🚀 Quick Start

```bash
# Install with semantic matching support
pip install propact[semantic]

# Use semantic matching
propact README.md --openapi api.json --base-url "https://api.example.com"
```

## 🔧 Advanced Usage

### Custom Model

```python
from propact.matcher import create_matcher

# Use a different sentence transformer model
matcher = create_matcher(model_name="all-mpnet-base-v2")
```

### Top-K Results

```bash
# Get top 5 matches instead of default 3
propact README.md --openapi api.json --base-url "https://api.example.com" --top-k 5
```

### Confidence Threshold

```python
from propact.matcher import EndpointMatcher

matcher = EndpointMatcher()
matches = await matcher.match(md_content, spec, top_k=3)

# Filter by confidence
high_confidence = [m for m in matches if m['score'] > 0.7]
```

## 🔍 Matching Algorithm

```python
# 1. Extract intent
intent = extract_intent(markdown_content)

# 2. Generate embeddings
intent_embedding = model.encode(intent)
endpoint_embeddings = model.encode(endpoint_descriptions)

# 3. Compute similarities
similarities = cosine_similarity(intent_embedding, endpoint_embeddings)

# 4. Rank and return top-k
top_matches = rank_similarities(similarities, k=3)
```

## 🎨 Integration with openapi-llm

When using specs generated by [openapi-llm](https://github.com/Superflows-AI/openapi-llm), Propact leverages LLM-enhanced descriptions for even better accuracy:

```json
{
  "summary": "Create user",
  "description": "Create a new user account",
  "x-llm-intent": "Register a new user with email verification and profile setup"
}
```

## 🛠️ Configuration

Add to your `pyproject.toml`:

```toml
[tool.propact.semantic]
model = "all-MiniLM-L6-v2"
top_k = 3
confidence_threshold = 0.3
enable_llm_enhancement = true
```
